# 川柳SNSアプリ「川柳なう」- Claude Code 指示文

## プロジェクト概要

BeRealにインスパイアされた、川柳版SNSアプリのプロトタイプを作成する。 1日1回ランダムなタイミングで通知が届き、5分以内に川柳を詠んで投稿するサービス。

---

## 技術スタック

- **フレームワーク**: Next.js（App Router）  
- **スタイリング**: Tailwind CSS  
- **状態管理**: Zustand（またはReact useState/useContext）  
- **データ**: ローカル状態のみ（外部DBなし。プロトタイプのためモックデータを使用）  
- **フォント**: Google Fonts の筆文字系フォント（例：`Kaisei Opti`, `Shippori Mincho`）

---

## 画面構成

### 1\. ホーム画面（投稿画面）

**画面を開いた瞬間に5分のカウントダウンを開始する。**

#### カウントダウン表示

- 画面上部に残り時間を大きく表示（例：`04:32`）  
- 残り1分を切ったら赤く点滅  
- 0になったら投稿フォームを無効化し、「時間切れ」表示にする  
- 時間切れになると友達の投稿も閲覧不可になる（「今日の句は終わりました」メッセージ）

#### 川柳作成UI

**2つのモードを切り替え可能：**

##### モードA：パーツ選択モード（デフォルト）

- 5音・7音・5音の3つのスロットをそれぞれ独立して埋める  
- 各スロットに「パーツ候補」がランダムで表示される（タップで選択）  
- 選択済みのパーツは別の候補に差し替えも可能（候補をシャッフルするボタンあり）  
- 音数のバリデーション：  
  - 各スロットに入れたパーツの合計音数がぴったり5・7・5になっていないと投稿ボタンが押せない  
  - ズレている場合は「音が合っていません（現在X音）」とリアルタイムでエラー表示

パーツのデータ構造（例）：

{

  "id": "p001",

  "text": "春風",

  "reading": "はるかぜ",

  "mora": 4

}

パーツリスト（mora=音数）をあらかじめ用意し、毎回ランダムに表示する。 以下を含む最低50個以上のパーツを定義すること：

- 季語・自然系（春風、夕暮れ、桜、月明かり、雪解け、秋風、入道雲、蛍、霜、波音、etc）  
- 動詞・形容詞系（揺れる、消えた、あざやか、淡い、ひとりぼっち、迷ってる、etc）  
- 助詞・接続系（が、は、に、を、と、でも、だけど、なのに、ながら、etc）  
- 感情・日常系（なんとなく、ねむい、うれしい、つらいな、etc）  
- 固有名詞・現代語系（スマホ、コンビニ、ライン既読、etc）

##### モードB：テキスト直接入力モード

- テキストエリアに自由入力  
- スラッシュ区切りで入力（例：`春風が／桜散りゆく／夕暮れに`）  
- 音数バリデーションはモードAと同様に実施

#### 投稿ボタン

- 5・7・5が揃って初めてアクティブになる  
- 投稿後はホーム画面が「友達の投稿一覧」に切り替わる

---

### 2\. 友達の投稿一覧画面

- 投稿完了 or 5分以内のみ閲覧可能  
- 時間切れの場合は「今日の句はもう終わりました。また明日。」とだけ表示

#### 投稿カード

- 筆文字フォントで川柳を大きく表示（縦書きを推奨）  
- 投稿者の名前・アイコン（プロトタイプ用モックユーザーのもの）を表示  
- 投稿時刻を小さく表示

#### リアクション機能

各投稿カードに以下のリアクションUIを用意：

1. **合格スタンプ**  
     
   - 赤い丸に「合格」と書かれたスタンプ風ボタン  
   - タップするとカードに貼り付くアニメーション  
   - 既にスタンプ済みの場合は押せない（またはトグル）

   

2. **赤ペン添削**  
     
   - 「赤ペンで添削する」ボタンを押すとテキスト入力欄が開く  
   - 入力テキストは赤ペン風のフォント・赤色で表示  
   - 例：「これ昨日の俺やないかい」「ガチ草」  
   - 添削コメントは投稿カードに重なるように表示（手書きメモ風）

## デザイン方針

### テーマ

**「現代的な墨と紙」**。古典的な和の美意識をベースに、SNSらしいポップさを掛け合わせる。

### カラーパレット

- 背景：生成り色（`#F5F0E8`）または薄墨（`#F0EDE6`）  
- 文字：墨色（`#1A1A1A`）  
- アクセント：朱色（`#C0392B`）※赤ペン・スタンプに使用  
- サブ：藍色（`#2C4A7C`）

### フォント

- 川柳本文：`Kaisei Opti` または `Shippori Mincho`（Google Fonts）  
- UI全般：`Noto Sans JP`

### アニメーション

- カウントダウンのテンション演出（残り30秒は背景が微妙にざわつく）  
- 合格スタンプを押した時のスタンプ押印アニメーション（ガタっと傾いて貼り付く）  
- 投稿完了時の墨が広がるような演出

### レイアウト

- モバイルファースト（スマホ縦向きを基本）  
- 川柳は縦書きで表示

## 補足・注意事項

- ログイン機能は不要。画面を開いたら即「あなた」としてログイン済み状態にする  
- データはすべてインメモリ（localStorage可）。サーバー通信なし  
- 5分タイマーはページロード時点からスタート。リロードするとリセットされる（プロトタイプなので許容）  
- バリデーションは厳密に：5・7・5がぴったりでなければ投稿ボタンは非活性のまま  
- 赤ペンコメントは1投稿に複数つけられてよい
